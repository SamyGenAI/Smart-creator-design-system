# Brand Setup Questionnaire

Use this prompt set for consistent onboarding:

1. What is your brand name?
2. Write a one-sentence brand description.
3. Primary brand color (`#hex`)?
4. Canvas/background color (`#hex`)?
5. Preferred accent palette aliases (`#hex`) for:
   - accent-1
   - accent-2
   - accent-3
   - accent-4
   - accent-5
6. Primary font family?
7. Serif display font family?
8. Do you want to override corner radii (`glass-header`, `glass`, `card`, `pill`)?

If the user skips any optional values, keep existing values from `DESIGN.md`.
